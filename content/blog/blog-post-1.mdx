Today's the actual day one of this site. I sat down with Claude Code and got the foundation in place — tokens, layout, nav, theme toggle, the boring-but-necessary stuff that everything else builds on top of.

While I was at it, I also got Ollama running locally with Gemma 4 and Qwen2.5-Coder. The idea is pretty simple: not every task needs a frontier model. Formatting a blog post, cleaning up notes, running small repetitive skills — that's exactly the kind of work a smaller local model can handle fine. I'd rather save my Claude usage for the stuff that actually needs it (architecture decisions, debugging, real reasoning) and let local models pick up the routine work.

It's early, so there's not much to report yet beyond "it's set up and it works." But the split feels right: Claude Code for the things that need judgment, local models for the things that just need to get done.

**What's next:** homepage and about page, then the projects grid. I also want to actually wire up one of those local-model skills end to end — formatting this very post is the first candidate.
